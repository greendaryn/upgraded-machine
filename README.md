# Upgraded Machine

A C# project template for building modern applications.

## Getting Started

### Prerequisites

- [.NET SDK](https://dotnet.microsoft.com/download) (version X.X or higher)
- [Visual Studio](https://visualstudio.microsoft.com/) or [Visual Studio Code](https://code.visualstudio.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/greendaryn/upgraded-machine.git
   cd upgraded-machine
   ```

2. Restore dependencies:
   ```bash
   dotnet restore
   ```

3. Build the project:
   ```bash
   dotnet build
   ```

## Running the Project

```bash
dotnet run
```

## Testing

Run tests with:
```bash
dotnet test
```
